# 학교 분실물 서비스

이 프로젝트는 학교 분실물 찾기 및 등록을 위한 웹 페이지 시안입니다.

## 파일 구조

- `index.html` - 메인 페이지
- `코드/style.css` - 스타일시트
- `코드/app.js` - 간단한 자바스크립트 동작

## 실행 방법

1. `index.html` 파일을 브라우저에서 엽니다.
2. 로컬 웹 서버를 사용하려면 Visual Studio Code의 Live Server 확장이나 다른 간단한 서버를 사용할 수 있습니다.

## 설명

- 페이지는 분실물 서비스 소개와 샘플 분실물 목록을 보여줍니다.
- `app.js`는 현재 날짜와 샘플 항목 목록을 렌더링합니다.
